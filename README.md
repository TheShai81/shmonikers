# Shmonikers
A multi-player web game inspired from Monikers (aka Fishbowl) where users can select cards (or input their own!) and take turns describing and acting out the cards to their teammates.

## Skills

### Languages
- Python
- HTML
- JavaScript
- CSS
- Jinja2

### Tools/Frameworks
- Flask
- Ngrok (tunneling)

### Other Skills
- Full-stack web development with Flask
- RESTful routes
- Multiplayer game states
- HTTP routing & backend logic
- Session management
- Modular Python structure
- Object-Oriented Programming
- Graphic design in Google Drawings
